Follow these steps to configure and run the bot on your machine.

### 1. Configuration

Before launching the bot, you must set up the configuration file.

1.  Go to the `config/` directory.
2.  Open the `common.yml` file.
3.  Fill in the required fields, especially your `token`.

### 2. Install Dependencies

Ensure you have [Dart](https://dart.dev/get-dart) installed.
Run the following command at the root of the project to install the necessary dependencies:

```bash
dart pub get
dart bin/main.dart
```
